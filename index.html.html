Introduction

This Privacy Policy describes how I collect, use, and handle information through my personal application that interfaces with Instagram's messaging features.

Scope of Use

This application is strictly for personal use and only accesses my own Instagram account messages. No third-party accounts or data are accessed or processed through this application beyond what is necessary to view messages sent to my account.

Information Collection and Use

Information We Collect

The application only accesses:

  • Direct messages received on my personal Instagram account
  • Basic account information necessary for authentication
  • Message metadata (timestamps, sender information) associated with these messages

How We Use Information

The collected information is used solely for:

  • Viewing messages sent to my Instagram account
  • Personal message management
  • Authentication with Instagram

Data Storage and Security

  • No message data is stored outside of Instagram's platform
  • All communication with Instagram is conducted through official APIs
  • Standard security protocols are implemented to protect authentication credentials

Third-Party Access

  • This application does not share any message data with third parties
  • Message content is only visible to me as the account owner
  • The application does not interact with accounts other than my own

Data Retention

  • The application does not maintain any permanent storage of Instagram messages
  • Any cached message data is temporary and exists only during active use of the application

User Rights

As this is a personal-use application accessing only my own account, I maintain full control over all messages accessed through the application, including the rights to:

  • Access my message data
  • Delete messages from my account
  • Restrict or object to message processing
